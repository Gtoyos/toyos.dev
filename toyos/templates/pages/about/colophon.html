Colophon

In publishing, a colophon (/ˈkɒləfən, -fɒn/) is a brief statement containing information about the publication of a book such as the place of publication, the publisher, and the date of publication.

Toyos.dev is a static website built using Flask with the beauty of simplicity in mind, there is no need for anything more complex: HTML templates and a global CSS site theme. For deployment, a Waitress WSGI server is used over a Nginx reverse proxy. These is hosted in an AWS instance located in West Virginia (USA).

Toyos.dev was conceived in a night of February 2022 and finally made public on February 25th. See the changelog if you're intrested in what's new on this website.
