entrenamos.uy

Big sized distributed program which provides a platform for professors to create sport activities so users can enroll on them plus many additional social functionalities such as Following/Unfollowing users, mark activities as favourite, raffle prizes and much more.

This project was a great opportunity to gather experience in the following technologies and tools:

  • Java EE technologies (JSP, Swing, Apache CFX and JPA).
  • Core web technologies: HTML, CSS and JS.
  • Deployment techniques with Tomcat, Ant and Maven.
  • UML tools for software development

This was the final project for the programming V course of the University of the Republic (UDELAR) I've made along 3 other colleagues. You can find its source code on Github.
