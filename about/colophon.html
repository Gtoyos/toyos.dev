Colophon

In publishing, a colophon (/ˈkɒləfən, -fɒn/) is a brief statement containing information about the publication of a book such as the place of publication, the publisher, and the date of publication.

Toyos.dev is a static website handcrafted in HTML with the beauty of simplicity in mind. There is no need for anything more complex. There is a global CSS file which contains the site theme and a tiny javascript that is used for generating the random quote on top of the page (which could be disabled without further inconvenience). For serving it's documents, An Nginx web server hosted in an AWS instance located in West Virginia (USA) is used.

Toyos.dev was conceived in a night of February 2022 and finally made public on February 25th. See the changelog if you're intrested in what's new on this website.
