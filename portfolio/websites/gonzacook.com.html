gonzacook.com

A website for a small cooking business. gonzacook.com features a catalog of food products and a virtual kart for taking orders. The website is dynamically generated using Java Server Pages technology and is hosted on a tomcat server. The website is mobile-first oriented and relies on bootstrap for it's responsive components.

Despite using JSP, the core logic of the shopping kart is fully implemented in javascript, making the site interaction (cliking buttons, confirming forms, etc.) a seamless experience. JSP is used as a way of assembling the website templates and fast deployment.

If you are not at Montevideo im afraid they wouldnt be able to take your order. Nevertheless, feel free to take a peek or see it's source code on github.
